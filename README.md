# smile_extraction
 
# Feature
 Captures your best smile from the video.

# Requirements
* Python 3.8.10
* openvino 2021.4.2
* openvino-dev[pytorch] 2021.4.2
* Flask 2.2.2

# Usage
```bash
git clone https://github.com/yukiharada1228/smile_extraction.git
cd smile_extraction
git submodule update --init --recursive
pip install -r requirements.txt
python main.py
```
