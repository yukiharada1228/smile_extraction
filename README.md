# smile_extraction
 
# Feature
 Captures your best smile from the video.

# Requirements
* Python 3.10.8
* openvino-dev 2022.3.0
* Flask 2.2.2

# Usage
```bash
git clone --recursive https://github.com/yukiharada1228/smile_extraction.git
cd smile_extraction
pip install -r requirements.txt
python3 app.py
```
